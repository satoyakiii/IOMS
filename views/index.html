Inventory and Order Management System (IOMS)

A backend API with MongoDB for managing products, inventory levels, and customer orders.

About the Project

Assignment 3 - Part 1: This project implements a complete CRUD API using Node.js, Express.js, and MongoDB (native driver).

The system provides filtering, sorting, and projection capabilities for efficient data retrieval and management.

Basic Operations

  • GET /api/products - Get all products
  • GET /api/info - API information

Filtering Examples

  • /api/products?name=laptop - Filter by name
  • /api/products?minPrice=500 - Products above $500
  • /api/products?minPrice=500&maxPrice=2000 - Price range $500-$2000

Sorting Examples

  • /api/products?sortBy=price&order=asc - Sort by price (ascending)
  • /api/products?sortBy=price&order=desc - Sort by price (descending)
  • /api/products?sortBy=name - Sort by name

Projection Examples

  • /api/products?fields=name,price - Only name and price
  • /api/products?fields=name - Only product names

Combined Queries

  • /api/products?name=laptop&sortBy=price&order=desc&fields=name,price - Laptops, sorted by price (desc), show only name & price
