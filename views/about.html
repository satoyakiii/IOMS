Our Mission

The Inventory and Order Management System (IOMS) was created to solve real-world business challenges in inventory tracking and order processing. Our goal is to provide a robust backend system that automates routine operations while giving administrators full control over critical business data.

Technology Stack

IOMS is built with modern web technologies to ensure scalability, reliability, and ease of maintenance:

  • Node.js: Runtime environment for server-side JavaScript
  • Express.js: Web framework for routing and middleware
  • PostgreSQL: Relational database for core data (planned)
  • MongoDB: NoSQL database for logging and auxiliary data (planned)

Project Timeline

This project is being developed over 10 weeks as part of the Web Technologies 2 (Back End) course. Key milestones include:

  • Weeks 1-2: Project setup, Express.js basics, routing and forms
  • Weeks 3-4: API development and server-side logic
  • Weeks 5-6: Database integration (PostgreSQL and MongoDB)
  • Week 7: Backend deployment and environment configuration
  • Weeks 8-9: RESTful API completion and authentication
  • Week 10: Final testing and project demonstration
